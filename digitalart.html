Environment Art

Some naturey 3D models to sooth your soul.

Maple Trees

The dense leaves were achieved by assigning each polygon on the mesh to the whole UV space and then billboarding the leaf texture on each one.

Pine Trees

The pine trees are inspired from various species I observed during my time living in North Carolina.

Rocks

These rocks use an edge detection shader that help give it more defined edges.

Bushes

These bushes use the same technique as the maple trees, but have a softer texture and a lighter fresnel color to make them look puffier.

Character Design

Hacker bot creation process.

3D Wireframe

This model is optimized for use in a mobile game. It contains less than 5,000 triangles.

UVs and Rigging

In order to maximize both performance and detail, UVs are mirrored across the X axis, and multiple fingers share a single rig bone.

Final Model

With the texturing complete, the hacker bot model is ready to be imported into the game engine!
